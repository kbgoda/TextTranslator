# TextTranslator
Translates ".txt" Files from Source to Destination Language

Steps to run this python program:
1. Create a Google Cloud Platform Account
2. Create a JSON Access Key and download it into folder
3. Create a venv using "python3 -m venv TranslationAPI"
4. Activate venv using "source ../TranslationAPI/bin/activate"
5. Install dependencies needed using "pip3 install -r requirements.txt" in the terminal
6. Run the program using "python pytranslate.py"
7. Deactivate venv using "deactivate"
